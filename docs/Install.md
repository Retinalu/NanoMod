
## Prerequisites:
	* Python 2.7
	* python packages:
	        + h5py
		+ scipy
		+ numpy
	* BWA MEM
	* HDF5
	* rpy2
	* R package:
	        + ggplot2
	        + gridExtra
	        + scales

## Install:
	* git clone https://github.com/WGLab/NanoMod.
        * The script to be run is in bin/NanoMod.py
	
## Usage:
 For how to use them, please refer to [Usage](https://github.com/WGLab/NanoMod/blob/master/docs/Usage.md)
